GoLand 导入包时标红解决方法

在用GoLand 编辑go项目代码时

运行了如下命令：

go mod tidy  #增加缺少的包，删除无用的包

go mod vendor   #将依赖复制到 vendor 目录下

但GoLand中引入的包依然标红，看起来非常不舒服，解决方法如下：

MAC 上 打开 GoLand 点击左上角： GoLand -> Preferences -> Go -> Go Modules

启用 GO 模块集成, 然后重启 GoLand ，此时引入的第三方包就不标红了
