一、下载安装包

各系统版本Go安装包下载地址：https://golang.google.cn/dl/

二、安装

Linux 系统

1、删除原有的 /usr/local/go 文件夹，然后将刚刚下载的安装包解压缩到 /usr/local

$ rm -rf /usr/local/go && tar -C /usr/local -xzf go1.18.2.linux-amd64.tar.gz

2、安装成功验证

通过打开命令提示符并键入以下命令来验证您是否已安装 Go

$ go version

确认该命令打印已安装的 Go 版本信息。

Mac 系统

1、打开你下载的包文件，按照提示安装Go即可。

该软件包将 Go 发行版安装到 /usr/local/go。该软件包应将 /usr/local/go/bin 目录放在您的 PATH 环境变量中。您可能需要重新启动任何打开的终端会话才能使更改生效。

2、安装成功验证

通过打开命令提示符并键入以下命令来验证您是否已安装 Go

$ go version

确认该命令打印已安装的 Go 版本信息。

Windows 系统

1、打开您下载的 MSI 文件并按照提示安装 Go。

默认情况下，安装程序将安装 Go 到 Program Files 或 Program Files (x86)目录。您可以根据需要更改位置。安装后，您需要关闭并重新打开所有打开的命令提示符，以便安装程序对环境所做的更改反映在命令提示符中。

2、安装成功验证

在 Windows 中，单击开始菜单。在菜单的搜索框中，键入 cmd，然后按 Enter 键。在出现的命令提示符窗口中，键入以下命令来验证您是否已安装 Go

$ go version

确认该命令打印已安装的 Go 版本信息。

三、创建工作目录

如设置工作目录为 ~/alidata/www/go, 用于存放代码

mkdir ~/alidata/www/go"

# 工作目录下要求创建三个目录，src、pkg、bin，其中所有源码都必须放在src目录下

mkdir src pkg bin

四、设置环境变量

1、设置 $GOPATH & $GOROOT

vim ~/.profile 

# 在文件最后添加
export GOPATH="$HOME/alidata/www/go"  # 全局工作目录
export GOROOT="/usr/local/go"  # go语言工作链目录

保存退出

2、将 $GOPATH & $GOROOT 目录添加至PATH环境变量,方便之后调用生成的GO程序

vim ~/.profile 

export PATH=$PATH:$GOROOT/bin:$GOPATH/bin

3、使生效：

source ~/.profile
