一、脉诊的核心：九分为常，过则阴阳相隔

中医脉诊第一条铁律：

脉以九分为正常。太少，阳不及；太过，阳太盛。

正常脉象长度以手腕横纹为基准，九分（约一寸以内）为佳。一旦超过一寸，就不再是简单的"太过"，而是阴阳已经相隔，胃气已绝的危重信号。

脉象 方向 成因 病机
溢脉 向上超出一寸 阴脉太盛，阴盛顶阳，阳气被逼外散 阴阳相隔，胃气已无
覆脉 向下超出一寸 阴虚，脉管加长下坠 阴阳相隔，胃气已无

🔑 重点：溢脉往上走，覆脉往下走，统称"膈"——无论哪种，都是胃气消亡的脉象。

脉诊中阴阳的空间定位：

部位 阴阳属性 对应脏腑
寸脉（上焦） 阳 心、肺
尺脉（下焦） 阴 肝、肾
关脉（中焦） 脾胃 胃气所在，主土

吸入为阴（肝肾主之），吐出为阳（心肺主之），呼吸之间是脾胃在运转。一搭脉，阴阳虚实，尽在指下。

二、生气通天论：人为什么必须跟天地相通

《黄帝内经·素问》第三章开篇即说：

人之生，本于阴阳。天地之间，六合之内，五脏六腑经络，通通与天气相通。

所谓"六合"——天地上下左右四方。中医认为人不是孤立的机器，而是跟自然界气息相通的整体。

西医看形、看验血报告；中医认为人必须顺乎自然。"其身有五，其气有三，凡此数者，则邪气伤人"——邪气怎么伤人、人为什么短命，全在这几个数里面。

三、阳气的周天运行：一吸一吐，就是一个小宇宙

阶段 路径 要点
吸气 口鼻 → 督脉 → 沿脊椎下行 → 命门（第14椎） 天气入体，化生热能（类比氧气产热）
吐气 命门 → 任脉 → 从鼻或口排出 废气外出

🔑 肾为先天之本，因为肾直接与天气相通。这就是"生气通天"的真意——生命之气与天地之气相接。

小肠-大肠-肺的水液循环（极重要）

这段是理解"阳气为何能走表"的关键：

  1. 食物入胃咀嚼后进入小肠，小肠内火极旺——火来自心脏，靠肺像鼓风炉一样把热往下挤
  2. 食物残渣与水分进入大肠，火在上烧，水被气化蒸腾
  3. 这股水蒸气从大肠回到肺
  4. 肺主皮毛 → 全身上下皮肤的津液来自肺，肺里的气来自大肠

🔑 喝下去的水在胃里就被蒸化了；食物里自带的水（蔬菜、肉、米里的水分）要到大肠才完成第二次气化，回到肺后，阳气才能走表、卫外。

四、伤阳气的四大元凶：寒、暑、湿、气

原文核心句："阳气者，卫外者也。" 阳气本来升发在外，护卫体表，抵御寒邪。但四种情况会伤到它：

① 阴于寒——里寒盛，阳不固表

心与小肠的火不够（里寒），阳气无法上升到体表：

  • 起居如惊（坐立不安）
  • 神气乃浮（精神涣散）

🔑 阳不能固表，神就飘了。

② 阴于暑（热）——津液伤，纯热症

太热 → 津液被烧干 → 汗多、烦躁、喘咳、话多 → 阳明证

  • 津液耗尽后，胃与大肠呈现纯热之象
  • 病毒居于阳明（胃与大肠）
  • 治法：清掉燥屎，津液自回，热自退

🔑 张仲景《伤寒论》的阳明篇，就是从这里领悟出来的。

③ 阴于湿——头如裹，筋缩短

湿气太盛（久坐湿地）→ 伤诸阳之会（头为诸阳之会）→ 阳气上头被阻：

  • 头好像戴了帽子、被绳子勒住一样
  • 大筋缩短、肌肉萎缩

🔑 湿伤阳 → 软短为拘（拘挛）→ 大筋软短，肌肉萎缩。把湿邪去掉，阳气就回来了。

④ 阴于气——气伤则肿

情形 表现 先后顺序
形先伤 → 气后伤 先痛后肿（如撞伤） 形→气
气先伤 → 形后变 没受伤却全身浮肿、外形改变 气→形

🔑 看到一个人外形变了、肿了，就是气伤了。治法：开阳气的药，把水蒸化掉。

五、阳气受损的五大临床表现

1. 烦劳过度 → 阳气耗损 → 惊厥昏厥

阳气本应循环不息。过于操劳，阳气耗损，停滞积于下：

  • 不出汗
  • 眼睛看不到了（你能看到瞳孔，但看不到视力——视力靠阳气撑着）

🔑 五官功能全靠阳气：耳能听、鼻能嗅、口能尝、目能视。阳散则目不明。突发失明若器质无病，就是阳气短暂受阻，恢复阳气即可复明。

2. 大怒 → 气血逆乱 → 中风、心脏病

人生气需要大量能量，阳气整个往上冲：

  • 眼睛红、脸涨红
  • 血随气涌于头部 → 昏厥、中风、心脏病发作

🔑 阳气不足伤到筋，手会抖、不听指挥。你能指挥身体运动，靠的是"神"不是"形"——看到的手是形，能不能动是阳（神）的问题。

3. 汗出偏沮 → 肌肉萎缩

阳气受损，汗出不止 → 肌肉失养而萎缩。

临床上萎缩症患者，汗一直流、不容易止住。

4. 汗出见湿 → 生疣

"汗出见湿，乃生痤疿。"

汗出后未擦干，接触外界湿气（雨雾、潮湿环境），停在皮肤表面 → 化生油质 → 疣（中医称"疣"，西医说是滤过性病毒）

🔑 实用建议：运动出汗后立刻用毛巾擦干，不要让汗停在皮肤上。久坐的人每隔一小时站起来走走——臀部反复生痘，往往就是汗出遇湿。

5. 高粱厚味 → 足生大疔；劳汗当风 → 生痤疮

情形 结果 机制
吃太好（肥甘厚味） 足生大疔疮 积热内蕴
劳动出汗遇风寒 红疹、水泡、化脓 寒束肌表，郁而化热

六、阳气与生活起居：清晨运动，晚上收藏

原则 说明
✅ 清晨运动 助阳气升发（少林寺和尚清晨练易筋经，不是晚上）
❌ 晚上剧烈运动 扰动阳气，不利收藏
✅ 阳气柔和 养经，经络柔顺不抽筋
❌ 开阖失常 寒气乘虚入 → 生大偻（身体佝偻弯曲）

🔑 "阳气者，精则养神，柔则养筋。" 把阳气练得精纯，就有神；柔和，筋就不抽。

七、脉诊实操：怎么摸出虚实？

中医讲营卫：

位置 含义
卫 脉外 气，主防御
营 脉内 血，主濡养

摸法：

  • 脉外 → 摸动能、强度、弹性 → 柔和为佳，刚强为病
  • 脉内 → 摸血量与质地：
    • 🧅 实脉 = 葱白（中间实）
    • 🧅 虚脉 = 葱叶（中间空）

🔑 闭眼摸不出葱白和葱叶的区别，脉诊这关还得练。但原理就这么简单：实脉按下去是满的，虚脉按下去是空的。

八、阳气不行的终极后果：褥疮

中风病人长期卧床不动 → 阳气不行 → 阴气累积 → 湿热蕴结 → 组织溃烂

这就是褥疮（中医称"破"）。有时溃烂面积大到可以"开门"，里面全烂了。

🔑 核心病机：阳气停滞 → 阴气积聚 → 湿热内生 → 化腐成脓。护理关键：勤翻身、勤活动，让阳气动起来。

写在最后

这堂课信息量极大，但底层逻辑就一条：

阳气是人体的发动机。它在，五官就灵，肢体就动，皮肤就固；它散了，目不明、耳不聪、筋不用、疮疖生。

护住阳气，说到底四件事：

伤害 对策
🥶 里寒伤阳 别让自己太冷
🔥 暑热伤津 别让自己太热
💧 湿邪困阳 别让自己太湿
😤 烦劳耗阳 别让自己太累

顺乎自然，日出而动，日落而息。古人说的这些话，放到今天依然是最朴素的健康真理。
