所有中医理论的源头，都在《黄帝内经》。搞懂五色、五味、五脏的对应关系，你不用背药方，看一眼脸色就知道病在哪、该往哪治。

一、先记住这张最基础的对应表

五脏 五色 五味 五畜 五谷 五果 五音 河图数
肝（木） 青 酸 鸡 粳（李） 李 角 8（阴之极）
心（火） 赤 苦 羊 黍（红米） 杏 徵 7（阳数）
脾（土） 黄 甘 牛 稷（小米） 枣 宫 5（中数）
肺（金） 白 辛 马 稻（白米） 梨 商 9（阳之极）
肾（水） 黑 咸 彘（黑猪） 豆 栗（坚果） 羽 6（阴数）

重点：这张表不是死记的，是从菜市场就能推出来的。

二、相生相克不是算命，是判断疗效的尺子

很多人以为"克"是克死、克制。错。五行的"克"是制衡，不是消灭。 肺金克肝木，是肺的筋去制约肝木，不让它疯长。五脏之间互相拉力均衡，人才相安无事；一旦失衡，就是病的根源。

相生 = 吉，逆克 = 凶。 这是望诊判断疗效的核心逻辑：

  • 患者脸色青（肝病）来找你 → 治了两周变红（木生火）→ ✅ 方向对了，继续用药
  • 患者脸色青 → 治完变黑（水克木）→ ❌ 你治错了，是医之过
  • 患者脸色黑（肾病）来找你 → 治完变青（水生木）→ ✅ 在好转；变白（金克水）→ ❌ 恶化了

重点：当令的季节出现当令的颜色 = 正常。春天见青色是好的，春天见白色就不对。用颜色的流转方向，就能判断你的治疗是在救人还是在害人。

三、跑一趟菜市场，五味归经全懂了

老师说得很直白：你不用尝百草，看一眼菜市场就够了。

菜场实物 对应脏腑 原理
青菜叶（青）→ 菜梗（白）→ 黄瓜（黄）→ 香菇（黑）→ 番茄（红） 肝→肺→脾→肾→心 五色全了
味淡的蔬果 入气分（三焦、水道） 气主运行
味厚的肉类、中药 入血分 血主收藏

最经典的例子：柠檬。 皮是青的，肉是酸的——摆明了告诉你"我入肝"。酸味滋润肝脏、同时排泄肝毒。切片带皮泡水喝，一点都不酸，随时随地养肝。

重点：吃酸一定要吃天然的酸（柠檬、酸奶），不是人工果酸。任何东西太过都不好，适度才是养。

四、五脏各有"开窍"，望诊有迹可循

脏 开窍于 病象 临床速判
肝 目 易惊骇、抽筋 眼睛后方是肝经所过
心 耳 影响五脏（血行全身） 天地之气从耳通心
脾 口 口甜 = 脾虚 甜味本应藏在脾内，外溢说明阳虚固不住
肺 鼻 病在背 白色分两种：枯骨白 = 无神（病），光亮白 = 有神（好）
肾 二阴 大小便都归肾管 大便溏泄日数次 = 肾阳虚；大小便俱闭 = 肾衰竭

特别实用的一招：看头发。 有气的人，头发是油的（油 = 气的外现）；气要走的时候，头发突然变干、蓬起来——神已经离开了。

重点：脾病口甜、肾病看二便、肺病看白色有没有神、肝病看眼睛和筋。一看就知道，不用等摸脉。

五、数的秘密：河图数字就是五行密码

河图口诀：天一生水，地六成之；天二生火，地七成之；天三生木，地八成之；天四生金，地九成之；天五生土，地十成之。

  • 肝数8 → 阴之极 → 人体最深的脏
  • 心数7 → 阳数 → 火从天来
  • 脾数5 → 居中
  • 肺数9 → 阳之极
  • 肾数6 → 阴数

看到数字就知道五行属性，马上知道这药往哪走。八角、三七、五香……中药命名里全是五行的影子。

六、人工盐 vs 天然盐：一个被忽视的治病细节

治肾病要用咸味引药入肾，但咸味指的是天然海盐、岩盐，不是人工精制盐。

人工盐颗粒极细，吸收深入体内，会产生成瘾性（所以老外吃牛排拼命撒盐，越吃越多）。天然海盐不会成瘾——谁没事拿着盐当零食吃？

重点：中医说"咸味入肾"是有前提的——必须是天然的咸。人工盐不但不能治病，还会积水、水肿。这就是为什么西医说"不能吃盐"，中医说"肾病要吃盐"——用的根本不是同一种盐。

七、临床有多快？看完脸色处方就开好了

冬天来个患者，脸色暗暗的（肾黑，正常）。突然两个眼圈很红——心脏病，看完了。不用摸脉，不用问诊半天，处方已经写好了。病人还在那讲故事，你已经知道该开什么药了。

这就是《黄帝内经》给你的底气："善为脉者，察五脏六腑，一从阴阳表里雌雄之际。"

重点：学完神农本草、伤寒金匮之后，望诊+处方一气呵成。病人觉得你在听他讲半天，其实你30秒就看完了。这才是中医训练5000年的真功夫。

写在最后

《素问·阴阳应象大论》说：

"阴阳者，天地之道也，万物之纲纪，变化之父母，生杀之本始，神明之府也。治病必求于本。"

五色、五味、五音、五谷……天地万物都能归纳进五脏。你不需要死记硬背，只要理解了"正常是什么样"，"不正常长什么样"，一通百通。

知道什么是常，就知道什么是病。知道相生的方向，就知道治疗该往哪走。

这就是中医最底层的逻辑——简单、直接、有效。

非其人勿教，非其真勿授，是谓得道。
