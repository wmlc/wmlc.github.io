Git 重新添加已被忽略过文件(夹)

  • 移除 .gitignore 中的相关条目
  • 将文件添加回 Git 的索引
  • 提交更改

移除 .gitignore 中的相关条目

首先，你需要从 .gitignore 文件中删除相关的条目，这样 Git 才会重新考虑跟踪这些文件。

将文件添加回 Git 的索引

然后，你需要将这些文件重新添加到 Git 的索引中。可以使用 git add <file> 命令来做到这一点。 如果是目录，可以用递归的方式添加：

git add -A <directory>

注意，使用 -A 参数会将目录下的所有更改（包括新文件、已修改文件以及已删除文件）都添加到索引中。如果你只想添加特定的新文件，可以逐个添加。

提交更改

最后，你可以使用 git commit 命令来提交这些文件到本地仓库。
