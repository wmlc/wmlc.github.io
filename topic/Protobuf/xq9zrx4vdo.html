各类型注意点

  • 1、bytes

bytes 类型的字段 在 不同语言中表现不一样，如 c# 是 ByteString 而 PHP 是 string , 因此在传输 二进制数据时 PHP 会自动转为 base_64 格式，而 c# 则不会； 两种语言之间对接数据时要进行数据转码。
