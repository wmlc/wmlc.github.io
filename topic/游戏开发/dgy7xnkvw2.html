游戏类型的划分依据

游戏类型的划分主要基于以下几个维度：

  • 玩法机制（Gameplay Mechanics）：如操作、战斗、建造、解谜等。

  • 玩家互动方式：单人、多人合作或对抗。

  • 游戏节奏与视角：如回合制、即时制，第一人称、第三人称等。

  • 目标驱动类型：以胜负为目标，或以探索、收集、养成为主。

例如，MOBA（Multiplayer Online Battle Arena）是一种强调团队合作和策略的实时对战游戏，玩家控制具有独特技能的英雄，与队友协作对抗另一支队伍，目标通常是摧毁敌方基地。而MMO（Massively Multiplayer Online）则是大型多人在线游戏，玩家在一个持续运行的虚拟世界中扮演角色，与其他玩家互动、探索、战斗、社交。SLG（Simulation Game 或 Strategy & Simulation Game）则是策略模拟类游戏，玩家通过策略部署、资源管理、领地建设等方式发展自己的势力或文明。

常见的游戏类型及示例

以下是一些常见的游戏类型及其代表作品：

类型 全称 说明 代表游戏
FPS First-Person Shooter 第一人称射击游戏，玩家以主角视角进行射击战斗。 《使命召唤》、《CS:GO》
TPS Third-Person Shooter 第三人称射击游戏，玩家从主角背后视角进行射击战斗。 《堡垒之夜》、《战争机器》
RPG Role-Playing Game 角色扮演游戏，玩家扮演角色进行冒险和成长。 《巫师3》、《最终幻想》系列
ARPG Action Role-Playing Game 动作角色扮演游戏，结合动作元素的RPG。 《暗黑破坏神》系列
MMORPG Massively Multiplayer Online Role-Playing Game 大型多人在线角色扮演游戏。 《魔兽世界》、《剑网3》
RTS Real-Time Strategy 即时战略游戏，玩家实时进行资源管理和战斗部署。 《星际争霸》、《帝国时代》
SLG Simulation Game / Strategy & Simulation Game 策略模拟类游戏，强调策略决策和资源管理。 《文明》系列、《三国志》系列
MOBA Multiplayer Online Battle Arena 多人在线战术竞技场游戏，强调团队合作和策略。 《英雄联盟》、《Dota 2》
AVG Adventure Game 冒险游戏，注重故事情节和探索。 《神秘海域》、《生化奇兵》
ACT Action Game 动作游戏，强调玩家的操作和反应能力。 《鬼泣》、《忍者龙剑传》
STG Shooting Game 射击游戏，玩家控制角色进行射击战斗。 《雷电》、《合金弹头》
FTG Fighting Game 格斗游戏，玩家控制角色进行对战。 《街头霸王》、《拳皇》
SPG Sports Game 体育游戏，模拟各种体育运动。 《FIFA》、《NBA 2K》
RAC Racing Game 竞速游戏，模拟赛车竞速。 《极品飞车》、《马里奥赛车》
PUZ Puzzle Game 益智游戏，考验玩家的思维和解谜能力。 《俄罗斯方块》、《纪念碑谷》
MUG Music Game 音乐游戏，玩家根据节奏进行操作。 《太鼓达人》、《节奏大师》
SIM Simulation Game 模拟游戏，模拟现实或虚拟世界的各种活动。 《模拟人生》、《模拟城市》
Horror Horror Game 恐怖游戏，营造紧张和恐怖的氛围。 《生化危机》、《寂静岭》
Sandbox Sandbox Game 沙盒游戏，提供高度自由的游戏世界。 《我的世界》、《GTA》系列
Battle Royale Battle Royale Game 大逃杀游戏，玩家在不断缩小的安全区内竞争生存。 《绝地求生》、《Apex 英雄》
CAG Card Game 卡牌游戏：以卡牌为载体，基于卡牌对战或收集 《炉石传说》《万智牌》

其他游戏分类划分方式

按游戏模式划分

单机游戏（Single-player）

仅支持单人游玩。

代表：《塞尔达传说》《黑暗之魂》。

多人游戏（Multiplayer）

支持多人联机，可进一步细分：

本地多人：同一设备或局域网内对战（如《马里奥赛车》）。

在线多人：通过互联网联机（如《英雄联盟》《绝地求生》）。

合作游戏（Co-op）

玩家组队合作完成任务。

代表：《双人成行》《Overcooked》。

竞技游戏（Competitive）

强调玩家间对抗，常用于电子竞技。

代表：《CS:GO》《Dota 2》。

生存游戏（Survival）

玩家需在恶劣环境中生存并发展。

代表：《饥荒》《方舟：生存进化》。

按开发技术划分

2D 游戏

使用二维图形引擎开发。

工具：Unity（2D模式）、Godot、Cocos2d-x。

3D 游戏

使用三维图形引擎开发。

工具：Unreal Engine、Unity、CryEngine。

跨平台游戏

同时支持PC、主机、移动端等。

代表：《原神》《堡垒之夜》。

独立游戏（Indie Games）

由小型团队或个人开发，通常预算有限。

代表：《空洞骑士》《星露谷物语》。

按商业模式划分

买断制（Premium）

一次性付费购买，无额外收费。

代表：《艾尔登法环》《巫师3》。

免费游玩（Free-to-Play, F2P）

基础内容免费，通过内购或广告盈利。

代表：《王者荣耀》《Genshin Impact》。

订阅制（Subscription）

按月/年付费获取服务。

代表：《Xbox Game Pass》《EA Play》。

广告变现（Ad-based）

通过展示广告盈利，常见于休闲游戏。

代表：《糖果传奇》《地铁跑酷》。

按目标受众划分

硬核玩家（Hardcore Gamers）

针对深度体验和高难度挑战的游戏。

代表：《黑暗之魂》《只狼》。

休闲玩家（Casual Gamers）

简单易上手，适合碎片化时间。

代表：《动物森友会》《Flappy Bird》。

家庭/儿童游戏

适合全年龄段或特定年龄层。

代表：《马里奥系列》《乐高游戏》。
